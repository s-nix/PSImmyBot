## Getting Started

```powershell
dotnet build
pwsh -NoLogo -NoProfile -Command "Import-Module ./bin/Debug/net8.0/PSImmyBot.dll"
Get-Command -Module PSImmyBot
```

Use `Get-Help <CmdletName> -Detailed` for parameter-level docs after importing the module. The generated pages here provide fast cross-linking to the corresponding source files for deeper code exploration.
